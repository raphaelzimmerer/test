Modcon2 - RS485 Converter

  1. Home
  2. Produktdetails

Produkteigenschaften

  • • Gehäuse: Edelstahl AISI 316L, Endkappen aus Aluminium
  • • Antenne: Omnidirektionale, witterungsbeständige Antenne
  • • Datenspeicher: Integrierter Datenspeicher für Offline-Betrieb
  • • Serviceschnittstelle: Für manuelles Auslesen der Daten
  • • Komponenten: ersetzbar (z.B. Batterie, Elektronik, ...)
  • • Betrieb: Energiesparender Ultra Low Power Betrieb 24h/7d
  • • Qualität: Verwendung von Qualitätskomponenten
  • • Nachhaltigkeit: Nachhaltig durch modularen Aufbau und robustes Design
  • • Batterie: Langlebige Li-SOCL2 Markenbatterien
  • • Elektronik: „Made in Germany“ (Hard- und Software)
  • • Bedienung: Integrierter Ein-/Ausschalter zur einfachen Inbetriebnahme
  • • Optionales Zubehör: Premium Drucksonde

Technische Daten

  • • Übertragung: Wasserstand/-druck und Temperatur
  • • Datenlogging: Luftfeuchte, Luftdruck, Umgebungstemperatur
  • • Spannungsversorgung: Erfasst Zustand der Spannungsversorgung
  • • Genauigkeit Luftdruck : ± 1.7 hPa max.
  • • Genauigkeit Luftfeuchte: ±3 % RH
  • • Genauigkeit Temperatur: ± 1.25°C max.
  • • Umgebungstemperatur: -20...60°C
  • • Batterielaufzeit: > 5 Jahre @ 4 Samples pro Std.

LoRa Parameter

  • • Frequenz: 868 MHz
  • • Reichweite: Bis zu 15 km
  • • Stromverbrauch: 50 mA im Betrieb
  • • Protokoll: Class A, OTAA, FUOTA
  • • Antenne: LoRa 868MHz EU, externe Antenne
  • • Datenrate: Adaptiv oder statisch konfigurierbar
  • • Mikrocontroller: Moderner, energiesparender Mikrocontroller
  • • Sicherheit: Secure Key Management Hardware
  • • LoRa Version: 1.0.4

Optionale Features

  • • Anpassungen für Sonden mit anderem Datenausgang
  • • Individuelle Pegelrohradapter und Kabelabhängungen
  • • Kundenspezifische Firmware
  • • Anpassungen für weitere Anwendungen und Medien
  • • Kundenspezifisches Antennendesign
  • • LoRa class B Implementierung
  • • Support bei Planung, Installation, Integration, Service

Lieferumfang

  • • System: LoRa Pegelmesssystem PMS-100 mit Antenne
  • • Spannungsversorgung: Spannungsversorgung durch integrierte Batterien
  • • Dokumentation: Betriebsanleitung
  • • Software: JavaScript Payload Formatter
  • • Integration: Integration Guide

Dimension

  • • Abmessungen (LxD): 339x40 [mm]
  • • Gewicht: 0,45 [kg]
